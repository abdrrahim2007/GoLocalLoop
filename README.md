
# GoLocalLoop

**GoLocalLoop** is a web platform designed to bridge the gap between local businesses and customers. It allows local vendors to publish their products and services, making it easier for community members to find exactly what they need from electronics to local tradespeople right in their neighborhood.

##  Overview
- **Problem**: Customers often struggle to find specific products or professional services (like electricians or plumbers) close to their location.
- **Solution**: A centralized platform connecting businesses with clients to facilitate communication and local searching.

##  Tech Stack
- **Frontend**: HTML5, CSS3, Tailwind CSS.
- **Logic**: JavaScript (DOM manipulation and custom logic).
- **Storage**: LocalStorage for persistent user data.

##  Key Features
- **Home Feed**: 3-column layout with a flexible navbar and dynamic main section.
- **Marketplace**: Automated grid display for products with overflow-scroll descriptions.
- **Freelancer Directory**: Filterable directory for service providers with price range and skill selectors.
- **Secure Auth**: Custom hashing for passwords before LocalStorage registration.
